# Examples

Example outputs that demonstrate what good results look like from this plugin.

## What goes here

- Decision briefs, risk summaries, scorecards
- Complexity heatmaps and architecture review notes
- Pattern remix drafts
- Any output that sets the quality bar for what you actually want delivered

## How agents and skills use this folder

When agents produce output, these examples set format expectations and the quality bar. Real outputs that you found useful are the strongest signal.

- **`synthesis-brief-writer`** references these for tone, structure, and level of detail.
- **`pattern-remix-planner`** uses these for output calibration when generating drafts.

## File guidelines

- Use the output contract formats defined in `docs/output-contracts.md`.
- Real examples preferred over synthetic ones — agents learn more from actual deliverables.
- One example per file with a descriptive filename.

## Example filenames

- `sample-decision-brief-vpn-evaluation.md`
- `sample-complexity-heatmap-cloud-migration.md`
- `sample-pattern-remix-hub-spoke.md`
