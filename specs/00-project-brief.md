# Project Brief

## Working title

Claude Code package for infrastructure and network architecture decision workflows.

## Primary target

Claude Code local project usage.

## Secondary target

Cursor compatibility later, only after the Claude Code workflow is working locally.

## Core idea

Build a local, inspectable package that bundles subagents, skills, and lightweight hooks to support senior infrastructure engineering workflows.

## What the package should help with

- accelerate first drafts from prior proven work
- surface hidden complexity in proposed designs
- split large context across multiple agents before synthesis
- produce decision-ready artifacts rather than generic summaries
- preserve source references where possible
- make deep technical reasoning more legible to other engineers and stakeholders

## Ideal outcomes

- better design reviews
- faster decision briefs
- clearer architecture tradeoff analysis
- fewer hidden implementation surprises
- reusable AI workflows that can spread beyond a single engineer

## Primary workflows

1. Pattern Remix
2. Complexity Mapper
3. Context Sharding

## v1 recommendation

Make **Complexity Mapper** and **Context Sharding** the hero path.
Include **Pattern Remix** in v1, but do not overbuild it.

## Key quality bar

This package should reflect the judgment style of a senior cloud and network engineer who is good at finding hidden implementation complexity.
